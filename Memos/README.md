# Инструкция по компиляции/сборке (Build Guide)

В данном разделе приведен список поддерживаемых платформ и формат команды для компиляции проекта.

## Поддерживаемые платформы (Dist List)

|     ОС      | Архитектура | Значение GOOS | Значение GOARCH |
| :---------: | :---------: | :-----------: | :-------------: |
|   **iOS**   |    amd64    |     `ios`     |     `amd64`     |
|  **Linux**  |    amd64    |    `linux`    |     `amd64`     |
| **Windows** |    amd64    |   `windows`   |     `amd64`     |

## Команда для сборки

Для кросс-компиляции используйте следующий шаблон команды в терминале:

```bash
GOOS=<Название_ОС> GOARCH=<Название_Архитектуры> go build -o <название_файла.exe> <название_исходника.go>
```

### Пример использования

Для сборки под Windows:

```bash
GOOS=windows GOARCH=amd64 go build -o myapp.exe main.go
```
