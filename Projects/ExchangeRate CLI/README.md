# 💱 Currency Converter CLI

Микро-проект на Go для конвертации валют в реальном времени через внешнее API. Проект демонстрирует работу с HTTP-запросами, парсингом JSON и интерфейсом командной строки (CLI).

## 🚀 Функционал

- Получение актуальных курсов валют через сторонний API.
- Конвертация заданной суммы из одной валюты в другую.
- Удобный интерфейс командной строки с использованием флагов.
- Обработка ошибок (некорректные коды валют, проблемы с сетью).

## 🛠 Технологический стек

- **Язык:** Go (Golang)
- **Библиотеки:**
  - `net/http` — для сетевых запросов.
  - `encoding/json` — для десериализации ответов API.
  - `flag` — для обработки аргументов командной строки.
- **API:** [ExchangeRate-API](https://exchangerate-api.com) (или аналогичный).

## 📋 Техническое задание (ТЗ)

### 1. Входные параметры

Программа должна поддерживать следующие флаги при запуске:

- `-amount` (float64): Сумма для конвертации (по умолчанию: `1.0`).
- `-from` (string): Исходная валюта в формате ISO (по умолчанию: `USD`).
- `-to` (string): Целевая валюта в формате ISO (по умолчанию: `RUB`).

### 2. Алгоритм работы

1. **Инициализация:** Чтение и валидация флагов командной строки.
2. **Запрос:** Формирование GET-запроса к API для получения курса валюты `-from`.
3. **Парсинг:** Декодирование JSON-ответа в Go-структуру (`struct`).
4. **Расчет:** Вычисление итоговой суммы на основе полученного коэффициента.
5. **Вывод:** Отображение результата в формате: `[Сумма] [Из] = [Результат] [В] (Курс: [Значение])`.

### 3. Обработка исключений

- Вывод понятной ошибки, если API недоступно.
- Проверка на корректность введенной суммы (число должно быть больше нуля).
- Информирование пользователя, если введен неверный код валюты.

## 💻 Пример использования

```bash
go run main.go -amount=100 -from=USD -to=EUR
```

**Вывод:**
`100.00 USD = 92.45 EUR (Курс: 0.9245)`
